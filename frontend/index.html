Store Management Processor System

Important Notes:

  • Store IDs: Must exist in Store Master Data (format: RP0000X)
  • Required Fields: Each visit needs valid store_id, image_url array, and visit_time
  • Image Requirements: URLs must be accessible and in JPEG/PNG format
  • Time Format: Visit time should be in RFC3339 (e.g., 2024-03-15T10:00:00Z)
  • Validation: System validates store IDs before creating jobs
  • Processing: Includes perimeter calculation with simulated GPU processing (0.1-0.4s delay)
